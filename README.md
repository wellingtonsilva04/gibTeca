# gibTeca
Site para compra e venda de gibis e seus derivados.

Para usar o site siga as seguinte orientações:
Crie um banco de dados com as seguinte especificações:

define('DB_SERVER', 'localhost');
define('DB_USERNAME', 'root');
define('DB_PASSWORD', '');
define('DB_NAME', 'mydb');

O script para criação do banco está na pasta: db/mydb.sql
